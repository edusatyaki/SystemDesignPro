# System Design Hub

A comprehensive, interactive reference for 50 system design case studies. Built as a fully static Single Page Application (SPA) with no framework dependencies.

## Topics Covered (50 Case Studies)

Includes deep-dives on:
- Distributed Infrastructure (URL Shortener, Rate Limiter, CDN, Web Crawler)
- Messaging & Chat (WhatsApp, Slack, Discord, Email)
- Video Streaming (Netflix, YouTube, Spotify, Zoom)
- Geospatial (Uber, Tinder, Google Maps, DoorDash)
- Social Platforms (Twitter, Instagram, News Feed)
- E-Commerce (Amazon, Ticket Booking, Payment Gateway)
- Data Systems (Fraud Detection, Recommendation, Analytics)
- ...and many more

## Features

- Dark, glassmorphism UI design
- Interactive architecture accordion (layer by layer exploration)
- Microservice tab explorer with API endpoints
- Database schema viewer
- Fully responsive (desktop + mobile)
- Instant search & category filtering
